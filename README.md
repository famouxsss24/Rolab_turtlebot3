## 배경 · 목적
원격 제어나 GPS에 의존하는 로봇은 신호가 끊기는 순간 멈춘다. 이 프로젝트는 외부 인프라 없이도 로봇이 스스로 지도를 만들고 경로를 판단해 주행하도록 — **엣지에서 인식 → 판단 → 제어가 닫히는** 자율성을 목표로 한다.

## 아키텍처 & Flow
```
센서(LiDAR · 카메라 · IMU)
  → SLAM / 지도 작성
  → Nav2 전역·지역 경로계획 · 장애물 회피
  → 제어 (ROS2 노드)
  → Isaac Sim 시뮬레이션 검증 → 실기(TurtleBot3) 이관
```

## 기술 스택
- **ROS2** — 노드/토픽 기반 로봇 미들웨어
- **Nav2** — 자율 내비게이션 (경로계획 · 회피 · 복구)
- **Isaac Sim** — 물리 기반 시뮬레이션 검증
- **Python · Linux** — 노드 구현 · 실행 환경
